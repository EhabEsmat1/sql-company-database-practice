# SQL Company Database Practice

This repository contains SQL practice queries based on the Company_SD sample database.

## Skills Demonstrated
- SELECT / WHERE
- INNER JOIN / LEFT JOIN
- GROUP BY / HAVING
- Aggregate Functions
- Subqueries
- Interview-style SQL questions

## Included Queries
1. Employees in department 10
2. Employees with salary > 1500
3. Projects by location
4. Employees with department names
5. Employee count per department
6. Employees working on each project
7. Employees with dependents
8. Employees with project count
9. Highest salary in each department
10. Highest salary employee
11. Employees above average salary
12. Department managers
13. Employees working on all projects
